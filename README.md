# PDF_from_fullscreen_image
Some online readers do not allow to download the original file (pdf or other). The ["GoFullPage"](https://gofullpage.com) addon available for some browsers is able to save all the pages as one or more (very tall) PNGs. This repository allows to process these to rebuild an actual pdf document with pages extracted from the images.

## Requirements
Python, OCRmyPDF, convert

## How to use
Read the small Python script and tailor the first part to match your needs. Improvements welcome ;)
